## 수업 시작 시간의 사용

교시 시간 목록은 아래와 같이 사용됩니다.

### 1. 수업 계획 정보

수업 계획 정보 추가 시 저장된 수업 시작 시간 목록 중 하나를 선택할 수 있어요.
